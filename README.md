# mysqltojson

Dump sql to json file
Usage : 
	sqltojson -d yourDB -t yourTable
	--help/-h : help 
	-d : database 
	-t : table
Optional arguments :
	-host : hostname (default localhost)
	-u : user (default root)
	-p : password (default hello)
	-c : character set (default utf8)
	-o : output file(default table name)
